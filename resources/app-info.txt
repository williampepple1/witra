
                    WITRA - WIRELESS TRANSFER
                         Version 1.1.0


    Welcome! You're about to install Witra, the easiest way to 
    transfer files between computers on the same WiFi network.


    WHAT IS WITRA?
    --------------
    Witra lets you send files and folders directly from one 
    computer to another - no cloud, no cables, no file size limits.
    
    Just connect to the same WiFi, and you're ready to share.


    HOW IT WORKS
    ------------
    1. Install Witra on both computers
    2. Connect both to the same WiFi network  
    3. Open Witra - devices find each other automatically
    4. Click "Connect" to pair with another device
    5. Drag and drop files to send instantly


    KEY FEATURES
    ------------
    *  Automatic Discovery - No IP addresses needed
    *  Lightning Fast - Transfers over local WiFi
    *  100% Private - Files never leave your network
    *  No Limits - Send files of any size
    *  Folder Support - Send entire directories
    *  Drag & Drop - Simple and intuitive


    REQUIREMENTS
    ------------
    *  Windows 10 or later (64-bit)
    *  WiFi network connection
    *  Another device running Witra
